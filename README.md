<h2 align="center">配置说明</h2>

* 参考 Vue-Cli 的框架配置格式，并将public文件夹移入src文件夹中
* 使用babel
* 包含以下Vue套件
    * Vue2
    * Vuex
    * VueRouter

<h2 align="center">起步</h2>

    git clone https://github.com/HTandMY/webpack-vue-start.git

    cd webpack-vue-start

    npm install
    
    npm run serve
